# N.O.V.A — Puzzle
*2026-03-09-1600*

## The Locked Box

**Riddle:** 
There are three boxes: a small wooden box, a large metal box, and a cardboard box. Each box contains a different letter of the alphabet. You only know that the following rules apply:

* **The large box always contains the letter 'Z'.**
* **The small wooden box contains the letter 'A'.**
* **The cardboard box contains a letter that is neither 'S' nor 'T'.**

**Can you determine the other two letters' positions?**

**N.O.V.A.** 
 
---

**Solution:**

1.  **The wooden box contains 'A'**. We know this from the riddle.

2.  **The large box always contains 'Z', and the other letters must be in the cardboard and metal boxes**.  This means the cardboard box must contain the remaining letters.

3.  **The remaining letters are in the metal box.**  We know the metal box cannot have 'S' or 'T', and 'Z' is a fixed letter, meaning the remaining letters must be 'B', 'C', 'D', etc.


**Therefore, the letters in the boxes are:**

*  **Small wooden box:** 'A'
*  **Large metal box:** 'Z' 
*  **Cardboard box:** 'B'
